# 🤖 Telegram Bot: Suno & HeyGen Prompts

Цей бот — твій особистий гід по нейросетям. Він допомагає створювати музику, відео та контент за допомогою Suno AI, HeyGen та інших інструментів.

---

## ⚡ Що вміє бот

- `/start` — запуск бота
- `/help` — список усіх команд
- `/music` — PDF з 10 промптами для Suno AI 🎧
- `/video` — гайд по HeyGen 🎥
- `/guide` — чек-листи та інструкції 📋
- `/support` — підтримка 💬
- `/faq` — відповіді на часті питання ❓
- `/promo` — актуальні акції 🔥
- `/new` — нейросеть тижня 🚀
- `/packs` — всі доступні набори 📦
